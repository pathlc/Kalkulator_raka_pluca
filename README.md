# Kalkulator-raka-płuca
Kalkulator wzorców architektonicznych raka gruczołowego nieśluzowego płuca z oceną grade. 

https://pathlc.github.io/Kalkulator_raka_pluca/

Ten kalkulator ma charakter edukacyjno-pomocniczy.
Nie stanowi wyrobu medycznego ani porady medycznej.
Autor nie ponosi odpowiedzialności za decyzje kliniczne.
